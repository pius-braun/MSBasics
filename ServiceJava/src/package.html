ServiceJava Microservice
This file is part of the Demo project MSBasics - Microservices Basics
ServiceJava is a Microservce implemented in Java. It sends a counter every second to a Message Queue Interface.
This microservice is the Publisher side of the Message Queue.
It accepts a "reset" command via a REST interface.
The Message Queue service is provided by RabbitMQ. RabbitMQ is running as a docker image on localhost.
Steps to get RabbitMQ running:
  • Install Docker Desktop from https://hub.docker.com/
  • Start Docker Desktop
  • Pull the RabbitMQ docker image: "docker pull rabbitmq"
    • create a custom network: "docker network create msbasics-network"
  • Start RabbitMQ: "docker run -d --hostname msbasics-host --network msbasics-network --name rabbit-msbasics rabbitmq
Steps to get ServiceJava running:
  • Build image using the Dockerfile
  • start image using "runservicejava.cmd"
